# Malaria Detection System (AI & Deep Learning)

## 📌 Overview
This project is a deep learning-based system for detecting malaria from blood smear images using Convolutional Neural Networks (CNN).

## 🧠 Tech Stack
- Python  
- TensorFlow / Keras  
- CNN (Convolutional Neural Network)  
- NumPy, Matplotlib, Scikit-learn  

## ⚙️ Features
- Image preprocessing and augmentation  
- CNN-based image classification  
- Class imbalance handling using class weights  
- Model evaluation using confusion matrix  
- Real-time prediction using GUI (Tkinter)

## 📊 Output
- Multi-class classification of malaria parasite types  
- Achieved model performance (~84% accuracy)
